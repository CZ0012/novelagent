# Fantasy Demo Project

This example mirrors the minimum demo scenario from `docs/architecture.md`:

- 3-ish story entities centered on Lin Jin, Helian Ya, and the Silver Crow Society.
- One old bell tower location.
- One lineage secret that the POV character must not know yet.
- One foreshadowing item about an early bell.
- One item clue: half of a black wax seal.

Run:

```powershell
python -m apps.cli.main demo
```
